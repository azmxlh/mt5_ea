# MT5 EA

This repository manages Expert Advisors (EAs) for MetaTrader 5 (MT5).

## Overview

This repository contains the source code (MQL5) for automated trading programs (EAs) that run on the MT5 platform.

## List of EAs

| Filename | Description |
|---|---|
| `Experts/SwapTrendNanpinEA.mq5` | Swap Trend Nanpin strategy EA |
| `Experts/TrendGridV3.mq5` | Trend Grid strategy EA (V3) |

## Environment

- MetaTrader 5
- MQL5
