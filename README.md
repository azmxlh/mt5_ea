# MT5 EA

MetaTrader 5 (MT5) 用の Expert Advisor (EA) を管理するリポジトリです。

## 概要

このリポジトリには、MT5プラットフォーム上で動作する自動売買プログラム（EA）のソースコード（MQL5）が含まれています。

## EA一覧

| ファイル名 | 説明 |
|---|---|
| `Experts/SwapTrendNanpinEA.mq5` | スワップ・トレンド・ナンピン戦略 EA |
| `Experts/TrendGridV3.mq5` | トレンドグリッド戦略 EA (V3) |

## 環境

- MetaTrader 5
- MQL5
